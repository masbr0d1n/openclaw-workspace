🔑 KEY IMPLEMENTATION DETAILS:

UUID FILENAME:
──────────────
import uuid
video_uuid = str(uuid.uuid4())  # e.g., 546ed002-a0eb-4c13-be4e-2a75f9161594
video_path = f"{video_uuid}.mp4"

THUMBNAIL GENERATION:
──────────────────────
• FFmpeg extracts frame at 1 second (or mid-point if shorter)
• Saves as /uploads/thumbnails/{uuid}.jpg
• Resizes to 320x180 (16:9 aspect ratio)
• JPEG quality 85%

DURATION EXTRACTION:
──────────────────────
• FFmpeg probe extracts exact duration
• Stored in database as float (seconds)
• Displayed as "Xm Ys" format (e.g., "3m 45s")

OFFLINE PLAYBACK:
────────────────
• Video stored in /app/uploads/videos/
• Served via Next.js proxy: /api/videos/file/uploads/videos/{uuid}.mp4
• No YouTube API calls during playback
• Works without internet connection

YOUTUBE SEARCH:
───────────────
• yt-dlp search with query + category filter
• Returns up to 10 results (configurable)
• Includes: title, description, thumbnail, duration, channel, views
