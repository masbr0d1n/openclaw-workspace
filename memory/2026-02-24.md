## Key Technical Decisions

### Database
- Chose PostgreSQL over MongoDB
  - Reason: Relational capabilities, modern async support (asyncpg)
  - Decision validated: All CRUD operations working

### ORM
- SQLAlchemy 2.0 async
  - async/await throughout
  - Connection pooling
  - expire_on_commit=False for tests

### Authentication
- JWT with python-jose
- Bcrypt password hashing (direct, not passlib)
  - Reason: Passlib has 72-byte limit bug
  - Fix applied: Truncate passwords to 72 bytes before hashing

### Validation
- Pydantic v2 throughout
- ConfigDict for model configuration
- Email-validator dependency added

### Testing
- In-memory SQLite for speed
- Pytest with asyncio mode
- Coverage target: >80% (achieved 90%)

### Deployment
- Docker multi-stage ready
- docker-compose for development
- Health checks configured
